API de Galería Virtual

Backend para la plataforma de galería de arte virtual

Esta es la página de inicio de la API. Para ver la documentación completa, póngase en contacto con el administrador.

Rutas Principales

  • GET /api/obras - Lista de obras
  • GET /api/obras/destacadas - Obras destacadas
  • GET /api/obras/:id - Detalle de obra
  • GET /api/categorias - Lista de categorías
  • GET /api/artistas - Lista de artistas
  • POST /api/auth/login - Iniciar sesión
  • POST /api/auth/register - Registrarse
  • POST /api/consultas - Enviar consulta/oferta

Rutas Protegidas

Las siguientes rutas requieren autenticación mediante token JWT:

  • GET /api/auth/me - Información del usuario autenticado
  • POST /api/obras - Crear nueva obra
  • PUT /api/obras/:id - Actualizar obra
  • DELETE /api/obras/:id - Eliminar obra
